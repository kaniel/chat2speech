# chat2speech

> Based on the below articles and reference code. Thanks!

Usage

- Run `docker-compose up`
- Navigate to:
  - http://127.0.0.1:8888/ - Public room
  - http://127.0.0.1:8888/ANY_ROOM_NAME - A room with the name ANY_ROOM_NAM
